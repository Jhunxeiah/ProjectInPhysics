# Quiz Workflow

The quiz now opens directly.

## Student Workflow

1. Open `simulation.html` and enter Quiz Mode, or open `quiz.html` directly.
2. Answer each question.
3. When the quiz is complete, the final score appears on the page.
4. The final score appears immediately on the quiz screen.

## Notes

- Draft answers in simulation quiz mode still save locally so students can close and reopen the modal during the same browser session.
- The removed page now points users back to the menu.
